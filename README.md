# Mi Proyecto Web - Ing. Mávila Camus 🚀

Este es un proyecto sencillo de una página personal diseñado para practicar el flujo de trabajo en Git y GitHub, así como el diseño responsivo con HTML5 y CSS3.

## 📋 Características
- Diseño moderno con degradados y fuentes de Google.
- Tarjetas con efectos de elevación (hover).
- Estructura semántica HTML5.
- Adaptable a dispositivos móviles (Responsive Design).

## 💻 Instalación y Uso Local

Si deseas probar este proyecto en tu computadora, sigue estos pasos:

### 1. Clonar el repositorio
Abre tu terminal y ejecuta el siguiente comando para traer el código a tu máquina:
git clone https://github.com/mavilacamus/Proyecto-Web-Ejemplo.git


## 🌿 Gestión de Ramas

Para este proyecto se siguió una estrategia de ramificación para mantener la estabilidad del código principal:

* **`main`**: Es la rama principal que contiene la versión estable del proyecto. Aquí se encuentra la estructura base inicial del sitio web.
* **`mejora-estilo`**: Rama dedicada exclusivamente al diseño visual. En esta rama se implementaron:
    * Integración de Google Fonts (Poppins).
    * Diseño de tarjetas (cards) con efectos de elevación e interactividad.
    * Sistema de colores basado en degradados profesionales.
    * Ajustes de diseño responsivo para asegurar la visualización en dispositivos móviles.
